# SafeSteps

> **Learn cybersecurity one safe step at a time.**

SafeSteps is an interactive cybersecurity education platform that helps users build safer digital habits through practical learning modules instead of lengthy theory. It focuses on everyday online security, making cybersecurity approachable for students and beginners.

Developed as part of my cybersecurity summer training, the project explores how simple, hands-on experiences can improve digital awareness and encourage secure online practices.

## Features

* Password Health Check
* Spot the Scam (Phishing Awareness)
* Privacy Health Review
* Browser Safety Review
* Learning Hub with cybersecurity resources
* Digital Safety Support
* Progress tracking dashboard
* Responsive cybersecurity-inspired interface

## Tech Stack

* Next.js
* React
* TypeScript
* Tailwind CSS
* Framer Motion
* Lucide Icons
